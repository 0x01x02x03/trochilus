Trochilus
========

**A fast&free windows remote administration Tool**

Coded in C++ (using VS2010)

Features
---
* Support TCP/IP,HTTP,DNS
* Serilize Protocol
* Singled-Threaded
* File Manager
* Remote Shell
* Download & Execute
* Upload & Execute
* System Information

Compiling
---
Open the project in Visual Studio and click build.

Build
---
Run build.bat
